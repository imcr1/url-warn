URL Warning Extension

Protecting you from potentially harmful websites

About This Extension

URL Warning is a browser extension designed to help you browse the web safely. It checks websites against blacklists and whitelists to warn you about potentially harmful sites and suggest safer alternatives.

Features

  • Automatic checking of websites against customizable blacklists
  • Whitelist support for trusted websites
  • Ability to load lists from local files or remote URLs
  • Warning notifications for blacklisted sites
  • Suggestions for safer alternative websites
  • Option to leave unsafe sites immediately
